# Registro Diario Excel

Automatización de registro de gastos, ingresos y ahorros en Excel con Python.

¿Qué hace?
- Pide fecha, tipo de movimiento, categoría, especificación y monto
- Muestra los datos y pide confirmación antes de guardar
- Detecta automáticamente la última fila disponible sin sobreescribir datos
- Guarda directamente en el archivo Excel

Requisitos
- Python 3
- openpyxl (`pip install openpyxl`)

Uso
1. Colocá tu archivo Excel en la misma carpeta que el script
2. Cambiá el nombre del archivo en el código por el tuyo
3. Ejecutá el script con `python "python registro diario.py"`
